
# TG记账机器人

基于`nodejs`开发的TG记账机器人

## 安装部署

源码拉取到本地后，填写自己的机器人token，执行sql文件生成数据库结构即可部署。<br>
打包`npx webpack --config webpack.config.js`

## 实现功能

1️⃣操作人设置<br>
2️⃣入账/下发<br>
3️⃣设置汇率<br>
4️⃣z0查询欧意价格<br>
5️⃣+0/-0查询今日/昨日账单<br>
6️⃣账单+日期：如账单2024-06-06 = 查询指定年月日账单 <br>
7️⃣计算器功能（可设置开关）<br>
8️⃣上课下课群禁言<br>
